# Demo Script (3–5 minutes)

1. **Problem** (20s)
   - Agents can act, but actions are hard to audit and trust.

2. **What ReceiptPilot does** (30s)
   - Runs autonomous goal loop and emits verifiable receipts for each step.

3. **Live run** (2 min)
   - Call `POST /run` with a goal.
   - Show generated receipts (`goal_received`, `plan_created`, `action_executed`, `final_result`, `verification`).

4. **Trust layer** (45s)
   - Explain hash-chain linkage and tamper detection.
   - Map this to ERC-8004 proof narrative + autonomous agent judging requirements.

5. **Track fit** (30s)
   - Let the Agent Cook: autonomous loop.
   - Agents With Receipts: auditable receipt pipeline.
   - Agent Services on Base: exposed service endpoint and service architecture.

6. **Close** (20s)
   - Show repo, architecture diagram, and roadmap to onchain receipt anchoring.
